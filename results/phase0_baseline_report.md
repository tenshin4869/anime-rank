# Phase 0 ベースライン実験 新規レポート (データリーク修正・100作品版)

**実験日**: 2026-04-17
**予測対象**: 放送開始から「1ヶ月後」と「3ヶ月後」のGoogle Trendsによる人気スコア
**入力データ**: 放送開始からの**最初の3週間（第1〜3週）**のデータ（PV、GTスコア、編集数など）
**データセット**: 2015〜2024年に放送されたアニメ92作品（全100作品中GT利用可能データ）

> [!WARNING]
> 本レポートは、以前の「第4週」データを含めることによるデータリーク（1ヶ月＝約4週の予測に対する直接的な解答の混入）を修正し、純粋に**最初の3週間の初速のみ**を使って未来を予測した結果をまとめています。

---

## 📊 レポートの読み方（評価指標について）

このレポートでは、モデルの予測精度を以下の3つの指標で評価しています。

1. **RMSE (平均平方二乗誤差)**: 0 に近いほど優秀。実際のスコアとどれくらいズレているか。
2. **Spearman ρ (順位相関係数)**: -1 〜 +1。高いほど、人気ランキングの並び順を正確に当てられていることを意味します（0.6以上で実用レベル）。
3. **R² (決定係数)**: 最大1.0。データ変動をどれくらい説明できているか。

---

## 🎯 結果からわかること（主なインサイト）

第4週のデータを隠して（リークを排除して）再検証した結果、以下の結論がより強固なものとなりました。

> [!IMPORTANT]
> **最初の3週間だけで、「1ヶ月後」のピーク人気は完璧に予測できる**
> 第4週のデータ（予測対象日の直前）を削っても、時系列テストにおいて Ridge回帰モデル が **R² = 0.75 / Spearman ρ = 0.86** という極めて精度の高い予測を実現しました。「最初の3週間の検索初速」さえ測れば、そのアニメの1ヶ月後の評価は決まると言えます。

> [!TIP]
> **「3ヶ月後」の壁は依然として厚い**
> 「放送3ヶ月後」の予測は R² が 0.2前後 まで急落し、Spearman ρ も 0.5付近 に留まります。3週間の初速だけでは、中盤から後半にかけての口コミによる伸びや失速までは捉えきれないことがわかります。

> [!NOTE]
> **シンプルな線形モデル（Ridge回帰）の強さ**
> 全く変わらず、全てのターゲット・検証方法において、Random Forest や XGBoost よりもシンプルな Ridge回帰 が最高の精度を出しました。これは「序盤のWiki PVや検索数と、将来的な人気ピークはきれいに比例する」という非常に素直な市場特性を示しています。

## 🎯 まとめ
「第4週のデータを使って1ヶ月後を予測する」というチート（リーク）を排除しても、**第3週までのデータだけでRidge回帰モデルを用いれば、極めて高精度に1ヶ月後の人気ランキングを予測できる** ことが証明されました。
今後は「いかにして3ヶ月後のランキング精度を引き上げるか（口コミによるバズ等の別指標の導入など）」が焦点となります。
