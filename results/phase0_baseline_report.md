# Phase 0 ベースライン実験 レポート (1〜5ヶ月後 予測拡張版)

**実験日**: 2026-04-17
**予測対象**: 放送開始から「1, 2, 3, 4, 5ヶ月後」のGoogle Trendsによる人気スコア
**入力データ**: 放送開始からの**最初の3週間（第1〜3週）**のデータ（PV、GTスコア、編集数など）
**データセット**: 2015〜2024年に放送されたアニメ100作品

---

## 🎯 結果からわかること（主なインサイト）

「初速（最初の3週間）」だけでどこまで未来の人気を予測できるのか、5段階（約150日後まで）で検証を行いました。結果として、**「2ヶ月の壁」** が非常に明確に浮かび上がりました。

> [!IMPORTANT]
> **2ヶ月後までは極めて高精度に予測可能**
> 1ヶ月後（ρ=0.90 / R²=0.82）、2ヶ月後（ρ=0.79 / R²=0.59）と、初月の勢いはそのまま2ヶ月目まで持続するため、単純な Ridge回帰モデル でも実用レベルを超える極めて高い精度でランキングを予測できます。

> [!WARNING]
> **3ヶ月目からの急激な精度低下と、4ヶ月以降の予測限界**
> 3ヶ月後になると R²=0.25 まで急落し、グラフ（予測時点別精度）でも崖のように下落します。さらに **4ヶ月後・5ヶ月後になると R² がマイナス（完全にノイズ、予測不可能）** となります。これは「最初の3週間」の熱量だけでは、放送終了後の長期的な人気を持続している作品と、終わってすぐに忘れられる作品を区別できないことを証明しています。

## 🎯 次のステップ・考察
この実験により **「放送序盤（第1〜3週）のWikipedia/Googleトレンド動向さえ見れば、1〜2ヶ月後の順位は高精度で予測できるが、それ以降の長期的な人気維持は予測できない」** という明確なベースライン（限界）が実証されました。

Phase 1 以降では、この「3ヶ月目以降に生まれる差」を埋めるために、**ユーザーのエンゲージメント指標** や **コア層/ライト層の分離** といった新しいアプローチの必要性が強く裏付けられたと言えます。
